# 仓库管理系统 (Warehouse Manager)

一个 Android 仓库管理系统应用，用于管理商品编码和仓储位置，并提供基于位置的导航功能。

## 功能特性

### 1. 商品管理 (CRUD)
- **录入商品**：添加商品编码、名称、仓储位置描述
- **位置坐标**：支持手动输入或自动获取当前 GPS 位置
- **编辑商品**：修改已有商品信息
- **删除商品**：删除不需要的商品记录
- **搜索商品**：通过商品编码、名称或位置进行搜索

### 2. 导航功能
- **输入商品编码**：搜索并选择要导航的商品
- **获取手机位置**：实时获取当前 GPS 位置
- **距离计算**：显示当前位置与商品位置的直线距离
- **闪烁频率指示**：
  - 距离越近，中心点闪烁频率越快
  - 5 米以内：100ms 闪烁一次（最快）
  - 10 米以内：200ms
  - 20 米以内：350ms
  - 50 米以内：500ms
  - 100 米以内：800ms
  - 200 米以内：1200ms
  - 200 米以上：2000ms（最慢）
- **信号强度条**：可视化显示距离远近

## 技术栈

- **语言**：Kotlin
- **架构**：MVVM
- **数据库**：Room
- **位置服务**：Google Play Services Location
- **UI 组件**：Material Design 3
- **最低 SDK**：Android 8.0 (API 26)
- **目标 SDK**：Android 14 (API 34)

## 权限说明

应用需要以下权限：

- `ACCESS_FINE_LOCATION`：精确定位（导航功能必需）
- `ACCESS_COARSE_LOCATION`：粗略定位
- `ACCESS_BACKGROUND_LOCATION`：后台定位（可选）

## 构建说明

1. 使用 Android Studio 打开项目
2. 同步 Gradle 文件
3. 运行到模拟器或真机

## 使用说明

### 添加商品
1. 点击主界面右下角「+」按钮
2. 填写商品编码、名称、仓储位置
3. 点击「获取当前位置」自动填充 GPS 坐标，或手动输入
4. 点击「保存」

### 导航到商品
1. 在主界面搜索或找到目标商品
2. 点击商品的导航图标，或选择「查看/导航」
3. 允许位置权限
4. 观察中心点的闪烁频率和距离显示
5. 根据闪烁频率变化靠近商品位置

## 注意事项

- 首次使用需要授予位置权限
- 导航功能在户外 GPS 信号良好时效果最佳
- 室内环境可能需要 WiFi 定位辅助
